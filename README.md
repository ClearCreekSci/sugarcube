# sugarcube
Python API to interface to [PiSugar2 Power Manager](https://www.pisugar.com/products/pisugar2-raspberry-pi-zero-battery)

You might ask why we don't use the official [pisugar-python driver](https://github.com/PiSugar/pisugar-python/blob/main/pisugar.py). We're just obstinate, NIH and all that.


[PiSugar2 Docs](https://github.com/PiSugar/PiSugar/wiki/PiSugar-Power-Manager-(Software))
